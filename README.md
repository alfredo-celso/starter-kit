# 🚀 Ultimate Passwordless Starter Kit

This is a professional and scalable Starter Kit for modern SaaS applications, built with a clean architecture that separates business logic, database queries, and data transport.

## ✨ Key Features

- **Passwordless Authentication:** Secure access using 6-digit one-time passwords (OTP).

- **Internationalization (i18next):** Native multilingual support in both the web interface and sent emails.

- **Centralized Data Model:** Clean backend architecture isolating SQL in the Models layer (`models/`).

- **Security and Sessions:** Robust server session management with `express-session` and route guardians (`requireAuth`).

- **Real Emails:** Integration with Gmail and `nodemailer` for synchronous and translated delivery of security codes.

## 🛠️ Technologies Used

- **Backend:** Node.js, Express.js, Handlebars (HBS)
- **Database:** MySQL
- **Security:** Crypto, Express-Session
- **Services:** Nodemailer, i18next

# 🚀 Ultimate Passwordless Starter-Kit

Este es un Starter-Kit profesional y escalable para aplicaciones SaaS modernas, construido con una arquitectura limpia que separa la lógica de negocio, las consultas a bases de datos y el transporte de datos.

## ✨ Características Principales

- **Autenticación Passwordless:** Acceso seguro mediante códigos de 6 dígitos de un solo uso (OTP).
- **Internacionalización (i18next):** Soporte nativo multiidioma tanto en la interfaz web como en los correos electrónicos enviados.
- **Modelo de Datos Centralizado:** Arquitectura limpia en el Backend aislando el SQL en la capa de Modelos (`models/`).
- **Seguridad y Sesiones:** Gestión robusta de sesiones de servidor con `express-session` y guardianes de ruta (`requireAuth`).
- **Correos Reales:** Integración con Gmail y `nodemailer` para el envío síncrono y traducido de códigos de seguridad.

## 🛠️ Tecnologías Utilizadas

- **Backend:** Node.js, Express.js, Handlebars (HBS)
- **Base de Datos:** MySQL
- **Seguridad:** Crypto, Express-Session
- **Servicios:** Nodemailer, i18next
